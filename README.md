# Assignment Deadline Tracker Web App

A clean, production-ready Streamlit web application that helps teachers manage assignments and allows students to view upcoming deadlines with visual indicators for urgency.

## Features
- **Teacher Dashboard**: Create, view, and delete assignments.
- **Student Dashboard**: View all assignments, filter by subject, and see a visual countdown of days remaining.
- **SQLite Database**: Automatically initialized, lightweight storage.

## Installation
Ensure you have Python 3.10+ installed.

1. Clone or navigate to the project repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the App
Start the Streamlit server with:
```bash
streamlit run app.py
```
